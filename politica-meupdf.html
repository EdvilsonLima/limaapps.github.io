Política de Privacidade

Aplicativo Meu PDF
Última atualização: 05/06/2026

Esta Política de Privacidade descreve como o aplicativo Meu PDF trata as informações dos usuários.

Ao utilizar o aplicativo, você concorda com os termos descritos nesta política.

1. Coleta de informações

O aplicativo não exige criação de conta, login ou cadastro de usuário.

O desenvolvedor não coleta, armazena ou compartilha dados pessoais diretamente.

Os arquivos PDF utilizados no aplicativo permanecem armazenados apenas no dispositivo do usuário.

2. Arquivos e armazenamento

O aplicativo pode acessar arquivos PDF armazenados no dispositivo apenas para executar funcionalidades solicitadas pelo usuário.

  • Visualização de arquivos PDF
  • Organização de documentos
  • Compartilhamento de arquivos
  • Gerenciamento de PDFs

Nenhum arquivo PDF é enviado para servidores do desenvolvedor.

3. Uso de anúncios

O aplicativo utiliza o Google AdMob para exibição de anúncios.

O Google AdMob pode coletar automaticamente determinadas informações para fornecer anúncios personalizados ou não personalizados.

  • Identificadores do dispositivo
  • Dados de uso do aplicativo
  • Endereço IP
  • Informações aproximadas de localização

Para mais informações:

https://policies.google.com/privacy

4. Compartilhamento de dados

O desenvolvedor não vende nem compartilha arquivos PDF ou dados pessoais dos usuários.

5. Permissões

O aplicativo pode solicitar permissões relacionadas ao armazenamento do dispositivo para acessar arquivos PDF selecionados pelo usuário.

Essas permissões são utilizadas exclusivamente para o funcionamento do aplicativo.

6. Segurança

O aplicativo foi desenvolvido utilizando boas práticas de segurança e privacidade.

Os arquivos manipulados permanecem sob controle do usuário no próprio dispositivo.

7. Alterações nesta política

Esta Política de Privacidade pode ser atualizada periodicamente.
