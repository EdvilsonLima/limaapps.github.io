Política de Privacidade Aplicativo Meu PDF Última atualização: 05/06/2026 Esta Política de Privacidade descreve como o aplicativo Meu PDF trata as informações dos usuários. Ao utilizar o aplicativo, você concorda com os termos descritos nesta política. 1. Coleta de informações O aplicativo não exige criação de conta, login ou cadastro de usuário. O desenvolvedor não coleta, armazena ou compartilha dados pessoais diretamente. Os arquivos PDF utilizados no aplicativo permanecem armazenados apenas no dispositivo do usuário. 2. Arquivos e armazenamento O aplicativo pode acessar arquivos PDF armazenados no dispositivo apenas para executar funcionalidades solicitadas pelo usuário, como: * Visualização de arquivos PDF * Gerenciamento de documentos * Compartilhamento de arquivos * Organização de PDFs O aplicativo não envia arquivos PDF para servidores externos do desenvolvedor. 3. Uso de anúncios O aplicativo utiliza o Google AdMob para exibição de anúncios. O Google AdMob pode coletar automaticamente determinadas informações para fornecer anúncios personalizados ou não personalizados. Essas informações podem incluir: * Identificadores do dispositivo * Dados de uso do aplicativo * Endereço IP * Informações aproximadas de localização * Dados relacionados à interação com anúncios Para mais informações sobre como o Google utiliza dados: https://policies.google.com/privacy 4. Compartilhamento de dados O desenvolvedor não vende, compartilha ou transfere arquivos PDF ou dados pessoais dos usuários. Serviços de terceiros, como o Google AdMob, podem processar informações conforme suas próprias políticas de privacidade. 5. Permissões O aplicativo pode solicitar permissões relacionadas ao armazenamento do dispositivo para acessar arquivos PDF selecionados pelo usuário. Essas permissões são utilizadas exclusivamente para o funcionamento das funcionalidades do aplicativo. 6. Segurança O aplicativo foi desenvolvido buscando boas práticas de segurança e privacidade. Os arquivos manipulados pelo aplicativo permanecem sob controle do usuário no próprio dispositivo. 7. Crianças O aplicativo não é direcionado especificamente para crianças. Nenhuma coleta intencional de informações pessoais de crianças é realizada pelo desenvolvedor. 8. Alterações nesta política Esta Política de Privacidade pode ser atualizada periodicamente para refletir melhorias no aplicativo, alterações legais ou mudanças em serviços de terceiros. Recomendamos que o usuário revise esta página ocasionalmente. 9. Contato Em caso de dúvidas sobre esta Política de Privacidade: [limaapps@hotmail.com](mailto:limaapps@hotmail.com)
