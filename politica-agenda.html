Política de Privacidade

Aplicativo Agenda Recorrente
Última atualização: 19/03/2026

Este aplicativo respeita a sua privacidade. Esta Política de Privacidade explica como as informações são tratadas.

1. Coleta de informações

Este aplicativo não coleta, armazena ou compartilha dados pessoais diretamente.

No entanto, serviços de terceiros utilizados no aplicativo podem coletar informações automaticamente, como dados de uso e identificadores do dispositivo.

2. Uso de anúncios

Este aplicativo utiliza o Google AdMob para exibição de anúncios.

O AdMob pode coletar e usar dados para fornecer anúncios personalizados ou não personalizados, incluindo:

  • Identificadores de dispositivo
  • Dados de uso
  • Informações aproximadas de localização

Para mais informações, consulte a política de privacidade do Google:

https://policies.google.com/privacy

3. Funcionalidades do aplicativo

O aplicativo pode direcionar o usuário para aplicativos externos, como:

  • WhatsApp
  • Aplicativo de telefone (ligações)
  • Aplicativo de mensagens SMS

Essas ações são iniciadas apenas quando o usuário interage com o aplicativo.

4. Permissões

O aplicativo pode solicitar permissões necessárias para:

  • Realizar chamadas telefônicas
  • Enviar mensagens SMS

Essas permissões são utilizadas apenas para executar funcionalidades solicitadas pelo usuário.

5. Compartilhamento de dados

O aplicativo não compartilha dados pessoais diretamente. No entanto, serviços de terceiros como o AdMob podem coletar informações conforme descrito acima.

6. Segurança

O aplicativo não armazena dados pessoais do usuário.

7. Alterações nesta política

Esta Política de Privacidade pode ser atualizada periodicamente.
