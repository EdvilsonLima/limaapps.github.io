Política de Privacidade

Aplicativo BabySleep: Ruído Branco

Última atualização: 06/07/2026

Este aplicativo respeita a sua privacidade. Esta Política de Privacidade explica como as informações são tratadas pelo aplicativo BabySleep: Ruído Branco.

O BabySleep foi desenvolvido para reproduzir ruído branco, sons relaxantes e sons ambientes que podem ajudar pais, mães e responsáveis a criar um ambiente mais calmo para o sono do bebê.

1. Coleta de informações

Este aplicativo não exige criação de conta, login ou cadastro.

O desenvolvedor não coleta diretamente dados pessoais como nome, e-mail, telefone, endereço ou informações do bebê.

No entanto, serviços de terceiros utilizados no aplicativo podem coletar automaticamente determinadas informações técnicas e de uso, incluindo:

  • Identificadores do dispositivo, como ID de publicidade;
  • Dados de uso do aplicativo, como interações, tempo de uso e cliques;
  • Dados de desempenho, falhas e diagnóstico;
  • Endereço IP;
  • Informações aproximadas de localização, quando utilizadas por serviços de publicidade.

Esses dados são coletados principalmente por serviços de terceiros integrados ao aplicativo.

2. Uso de anúncios

Este aplicativo pode utilizar o Google AdMob para exibição de anúncios.

O AdMob pode coletar e usar dados para fornecer, limitar, medir ou personalizar anúncios, incluindo:

  • Identificadores de dispositivo;
  • Dados de uso;
  • Informações aproximadas de localização;
  • Dados de desempenho e diagnóstico.

Para mais informações, consulte a política de privacidade do Google:
https://policies.google.com/privacy

3. Uso das informações

As informações coletadas por serviços de terceiros podem ser utilizadas para:

  • Exibição de anúncios;
  • Medição de desempenho dos anúncios;
  • Análise de uso e funcionamento do aplicativo;
  • Diagnóstico de falhas;
  • Melhoria da experiência do usuário.

4. Funcionalidades do aplicativo

O BabySleep permite que o usuário reproduza sons relaxantes, ruído branco e sons ambientes, como chuva, secador de cabelo, aspirador, máquina de lavar, pássaros e cachoeira.

O aplicativo também pode oferecer recursos como:

  • Reprodução contínua de áudio;
  • Timer para desligamento automático;
  • Funcionamento sem internet para os sons disponíveis no app;
  • Reprodução com a tela bloqueada;
  • Reprodução em segundo plano;
  • Tema claro e tema escuro.

5. Reprodução em segundo plano

O BabySleep pode continuar reproduzindo áudio mesmo quando o usuário sai do aplicativo, bloqueia a tela ou utiliza outro aplicativo.

Para permitir essa funcionalidade, o aplicativo pode utilizar a permissão de serviço em primeiro plano para reprodução de mídia, como FOREGROUND_SERVICE_MEDIA_PLAYBACK.

Essa permissão é usada apenas para manter a reprodução de áudio iniciada pelo usuário. Enquanto o som estiver sendo reproduzido em segundo plano, o sistema Android pode exibir uma notificação de mídia indicando que o aplicativo está ativo e reproduzindo áudio.

O usuário pode pausar, parar ou ajustar a reprodução a qualquer momento pelo próprio aplicativo ou pelos controles de mídia do dispositivo.

6. Funcionamento offline

O BabySleep pode funcionar sem conexão com a internet para reprodução dos sons disponíveis no aplicativo.

Mesmo funcionando offline para reprodução de áudio, alguns recursos de terceiros, como anúncios, análise ou diagnóstico, podem exigir conexão com a internet quando estiverem presentes.

7. Permissões

O aplicativo pode solicitar ou utilizar permissões necessárias para o funcionamento adequado de seus recursos, incluindo:

  • Permissão para reprodução de mídia em segundo plano;
  • Serviço em primeiro plano para manter o áudio ativo;
  • Notificações, quando necessário, para exibir controles de mídia;
  • Acesso à internet, quando necessário para anúncios ou serviços de terceiros.

Essas permissões são utilizadas apenas para executar funcionalidades relacionadas ao funcionamento do aplicativo.

8. Armazenamento de dados

O aplicativo pode armazenar localmente no dispositivo algumas preferências do usuário, como:

  • Tema claro ou escuro;
  • Tempo definido no timer;
  • Último som reproduzido;
  • Preferências de uso do aplicativo.

Essas informações são usadas para melhorar a experiência dentro do aplicativo e não são enviadas diretamente aos servidores do desenvolvedor.

9. Compartilhamento de dados

O desenvolvedor não vende dados pessoais dos usuários.

O aplicativo não compartilha diretamente dados pessoais coletados pelo desenvolvedor. No entanto, serviços de terceiros utilizados no app, como o Google AdMob, podem coletar, processar e compartilhar informações conforme suas próprias políticas de privacidade.

10. Segurança

O aplicativo não armazena dados pessoais do usuário em servidores próprios do desenvolvedor.

Dados tratados por serviços de terceiros podem ser transmitidos por conexão segura, como HTTPS, de acordo com as práticas e políticas desses serviços.

11. Dados de crianças

O BabySleep é destinado ao uso por pais, mães e responsáveis que desejam reproduzir sons relaxantes para auxiliar na rotina de sono do bebê.

O aplicativo não solicita, não coleta e não armazena diretamente dados pessoais de crianças ou bebês.

Caso seja identificado qualquer tratamento indevido de dados pessoais de crianças, serão tomadas as medidas necessárias para remoção ou correção.

12. Direitos do usuário

Como o desenvolvedor não coleta diretamente dados pessoais em servidores próprios, não há conta interna para exclusão de dados.

O usuário pode gerenciar dados relacionados à publicidade e identificadores do dispositivo nas configurações do Android, incluindo a redefinição ou limitação do ID de publicidade.

13. Alterações nesta política

Esta Política de Privacidade pode ser atualizada periodicamente para refletir alterações no aplicativo, em serviços de terceiros ou em requisitos legais e de plataforma.

Recomendamos que o usuário revise esta página regularmente.
