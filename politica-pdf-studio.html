Política de Privacidade

Aplicativo PDF Studio: Ler e Editar PDF

Última atualização: 12/07/2026

Esta Política de Privacidade descreve como o aplicativo PDF Studio: Ler e Editar PDF trata as informações dos usuários.

Ao utilizar o aplicativo, você concorda com os termos descritos nesta Política de Privacidade.

1. Coleta de informações

O aplicativo não exige criação de conta, login ou cadastro de usuário.

O desenvolvedor não coleta, armazena ou compartilha diretamente dados pessoais como nome, e-mail, telefone ou endereço.

Os arquivos PDF utilizados no aplicativo permanecem armazenados no dispositivo do usuário, exceto quando o próprio usuário decide compartilhar, exportar ou salvar o arquivo por meio de funcionalidades do sistema ou de outros aplicativos.

2. Arquivos PDF e armazenamento

O aplicativo pode acessar arquivos PDF armazenados no dispositivo apenas para executar funcionalidades solicitadas pelo usuário.

Essas funcionalidades podem incluir:

  • Leitura e visualização de arquivos PDF;
  • Edição ou ajustes em documentos PDF, quando disponível;
  • Organização e gerenciamento de documentos;
  • Compartilhamento de arquivos PDF;
  • Exportação ou salvamento de arquivos modificados;
  • Abertura de documentos selecionados pelo usuário.

Nenhum arquivo PDF é enviado para servidores próprios do desenvolvedor.

O acesso aos arquivos ocorre localmente no dispositivo e apenas conforme a ação realizada pelo usuário dentro do aplicativo.

3. Uso de anúncios

O aplicativo pode utilizar o Google AdMob para exibição de anúncios.

O Google AdMob pode coletar automaticamente determinadas informações para fornecer, limitar, medir ou personalizar anúncios, incluindo:

  • Identificadores do dispositivo, como ID de publicidade;
  • Dados de uso do aplicativo;
  • Endereço IP;
  • Informações aproximadas de localização;
  • Dados de desempenho e diagnóstico.

Para mais informações, consulte a Política de Privacidade do Google:
https://policies.google.com/privacy

4. Uso das informações

As informações coletadas por serviços de terceiros podem ser utilizadas para:

  • Exibição de anúncios;
  • Medição de desempenho dos anúncios;
  • Análise de uso e funcionamento do aplicativo;
  • Diagnóstico de falhas;
  • Melhoria da experiência do usuário.

5. Compartilhamento de dados

O desenvolvedor não vende dados pessoais dos usuários.

O desenvolvedor não compartilha arquivos PDF dos usuários com servidores próprios.

Serviços de terceiros utilizados no aplicativo, como o Google AdMob, podem coletar, processar e compartilhar determinadas informações conforme suas próprias políticas de privacidade.

6. Permissões

O aplicativo pode solicitar permissões relacionadas ao acesso a arquivos, documentos ou armazenamento do dispositivo para permitir que o usuário selecione, abra, edite, salve, exporte ou compartilhe arquivos PDF.

Essas permissões são utilizadas exclusivamente para o funcionamento das funcionalidades do aplicativo e dependem das ações realizadas pelo próprio usuário.

7. Segurança

O aplicativo foi desenvolvido utilizando boas práticas de segurança e privacidade.

Os arquivos manipulados permanecem sob controle do usuário no próprio dispositivo.

Dados tratados por serviços de terceiros podem ser transmitidos por conexão segura, como HTTPS, de acordo com as práticas desses serviços.

8. Armazenamento local

O aplicativo pode armazenar localmente algumas preferências do usuário, configurações de uso ou informações necessárias para melhorar a experiência dentro do app.

Essas informações permanecem no dispositivo e não são enviadas diretamente para servidores próprios do desenvolvedor.

9. Dados de crianças

O aplicativo não é direcionado especificamente a crianças e não coleta intencionalmente informações pessoais de crianças.

Caso seja identificado qualquer tratamento indevido de dados pessoais de crianças, serão tomadas as medidas necessárias para remoção ou correção.

10. Direitos do usuário

Como o desenvolvedor não coleta diretamente dados pessoais em servidores próprios, não há conta interna para exclusão de dados.

O usuário pode gerenciar arquivos PDF diretamente no próprio dispositivo e pode excluir arquivos, preferências ou dados do aplicativo nas configurações do Android.

O usuário também pode gerenciar dados relacionados à publicidade nas configurações do dispositivo, incluindo a redefinição ou limitação do ID de publicidade.

11. Alterações nesta Política

Esta Política de Privacidade pode ser atualizada periodicamente para refletir mudanças no aplicativo, em serviços de terceiros ou em requisitos legais e de plataforma.

Recomendamos que o usuário revise esta página regularmente.
