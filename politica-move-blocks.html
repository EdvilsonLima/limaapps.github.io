Política de Privacidade

Aplicativo Move Blocks
Última atualização: 19/05/2026

Bem-vindo ao aplicativo Move Blocks. A sua privacidade é importante para nós. Esta Política de Privacidade explica como as informações podem ser coletadas, utilizadas e protegidas durante o uso do aplicativo.

Ao utilizar o aplicativo Move Blocks, você concorda com os termos desta Política de Privacidade.

1. Informações coletadas

O aplicativo Move Blocks pode coletar algumas informações automaticamente por meio de serviços de terceiros utilizados no app, incluindo:

  • Identificador do dispositivo
  • Endereço IP
  • Dados de uso do aplicativo
  • Informações sobre anúncios visualizados e interações com anúncios
  • Dados de diagnóstico e desempenho

O aplicativo não coleta informações pessoais como nome, CPF, endereço ou fotos, exceto quando fornecidas diretamente pelo usuário através de serviços externos.

2. Publicidade

O aplicativo utiliza os serviços de publicidade do Google AdMob para exibir:

  • Anúncios em banner
  • Anúncios premiados (Rewarded Ads)

Esses serviços podem utilizar cookies, identificadores de publicidade e tecnologias semelhantes para exibir anúncios mais relevantes aos usuários.

Para mais informações sobre como o Google utiliza os dados, acesse:

https://policies.google.com/privacy

3. Uso das informações

As informações coletadas podem ser utilizadas para:

  • Melhorar a experiência do usuário
  • Corrigir erros e falhas
  • Analisar desempenho do aplicativo
  • Exibir anúncios relevantes
  • Garantir a segurança e funcionamento adequado do app

4. Compartilhamento de dados

O aplicativo pode compartilhar informações limitadas com serviços de terceiros utilizados para:

  • Exibição de anúncios
  • Análise de desempenho
  • Funcionamento técnico do aplicativo

Os dados não são vendidos para terceiros.

5. Crianças

O aplicativo Move Blocks não é direcionado especificamente para crianças menores de 13 anos.

Caso seja identificado o envio involuntário de dados por menores de idade, as medidas adequadas serão tomadas para remoção dessas informações.

6. Segurança

Adotamos medidas razoáveis para proteger as informações coletadas contra acesso não autorizado, alteração, divulgação ou destruição.

No entanto, nenhum método de transmissão ou armazenamento eletrônico é 100% seguro.

7. Links para serviços de terceiros

O aplicativo pode conter anúncios ou links para sites e serviços externos. Não somos responsáveis pelas práticas de privacidade desses terceiros.

8. Alterações nesta política

Esta Política de Privacidade poderá ser atualizada periodicamente. Recomendamos que os usuários revisem esta página regularmente para verificar possíveis alterações.
