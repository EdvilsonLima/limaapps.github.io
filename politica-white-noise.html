Política de Privacidade Aplicativo White Noise Última atualização: 12/04/2026 Esta Política de Privacidade descreve como o aplicativo White Noise trata as informações dos usuários. Ao utilizar o aplicativo, você concorda com os termos descritos nesta política. 1. Coleta de informações O aplicativo não exige login, cadastro ou criação de conta. O desenvolvedor não coleta, armazena ou compartilha dados pessoais diretamente. No entanto, serviços de terceiros utilizados no aplicativo podem coletar determinadas informações automaticamente, incluindo: * Identificadores do dispositivo * ID de publicidade * Informações de uso do aplicativo * Dados de desempenho e diagnóstico * Endereço IP * Informações aproximadas de localização 2. Uso das informações As informações coletadas por serviços de terceiros podem ser utilizadas para: * Exibição de anúncios * Personalização de publicidade * Medição de desempenho * Análise de uso do aplicativo * Melhorias na experiência do usuário 3. Publicidade O aplicativo utiliza o Google AdMob para exibição de anúncios. O Google pode utilizar identificadores do dispositivo e outros dados para fornecer anúncios personalizados ou não personalizados. Para mais informações sobre como o Google utiliza dados: https://policies.google.com/privacy 4. Compartilhamento de dados O desenvolvedor não vende, transfere ou compartilha dados pessoais diretamente. Os serviços de terceiros utilizados no aplicativo podem processar informações conforme suas próprias políticas de privacidade. 5. Armazenamento de dados O aplicativo não mantém banco de dados próprio nem servidores para armazenamento de informações pessoais dos usuários. Dados eventualmente coletados por serviços de terceiros são processados conforme as políticas dessas plataformas. 6. Segurança As conexões utilizadas pelo aplicativo podem utilizar protocolos seguros, incluindo HTTPS, para comunicação com serviços externos. 7. Crianças O aplicativo não é direcionado especificamente para crianças. Nenhuma coleta intencional de informações pessoais de crianças é realizada pelo desenvolvedor. 8. Direitos do usuário Como o aplicativo não realiza coleta direta de dados pessoais, não existem mecanismos internos de exclusão de dados. Os usuários podem gerenciar permissões e redefinir identificadores de publicidade diretamente nas configurações do dispositivo Android. 9. Alterações nesta política Esta Política de Privacidade pode ser atualizada periodicamente para refletir melhorias no aplicativo, mudanças legais ou alterações em serviços de terceiros.
