Política de Privacidade

Aplicativo White Noise
Última atualização: 05/06/2026

Esta Política de Privacidade descreve como o aplicativo White Noise trata as informações dos usuários.

Ao utilizar o aplicativo, você concorda com os termos descritos nesta política.

1. Coleta de informações

O aplicativo não exige login, cadastro ou criação de conta.

O desenvolvedor não coleta, armazena ou compartilha dados pessoais diretamente.

Serviços de terceiros utilizados no aplicativo podem coletar determinadas informações automaticamente.

  • Identificadores do dispositivo
  • ID de publicidade
  • Informações de uso do aplicativo
  • Dados de desempenho e diagnóstico
  • Endereço IP
  • Informações aproximadas de localização

2. Uso das informações

As informações coletadas por serviços de terceiros podem ser utilizadas para:

  • Exibição de anúncios
  • Personalização de publicidade
  • Análise de uso do aplicativo
  • Melhoria da experiência do usuário
  • Medição de desempenho

3. Publicidade

O aplicativo utiliza o Google AdMob para exibição de anúncios.

O Google pode utilizar dados e identificadores do dispositivo para fornecer anúncios personalizados ou não personalizados.

Para mais informações:

https://policies.google.com/privacy

4. Compartilhamento de dados

O desenvolvedor não vende, transfere ou compartilha dados pessoais diretamente.

Serviços de terceiros podem processar informações conforme suas próprias políticas de privacidade.

5. Armazenamento de dados

O aplicativo não possui banco de dados próprio nem servidores para armazenamento de informações pessoais dos usuários.

6. Segurança

O aplicativo utiliza boas práticas de segurança e pode utilizar conexões seguras (HTTPS) para comunicação com serviços externos.

7. Crianças

O aplicativo não é direcionado especificamente para crianças.

Nenhuma coleta intencional de informações pessoais de crianças é realizada pelo desenvolvedor.

8. Direitos do usuário

Como o aplicativo não realiza coleta direta de dados pessoais, não existem mecanismos internos de exclusão de dados.

Os usuários podem redefinir identificadores de publicidade diretamente nas configurações do dispositivo Android.

9. Alterações nesta política

Esta Política de Privacidade pode ser atualizada periodicamente para refletir melhorias no aplicativo, alterações legais ou mudanças em serviços de terceiros.
